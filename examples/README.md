## Regenerate arrhythmia examples

```bash
pip install -e .
python scripts/generate_arrhythmia_examples.py
```

This script:
1. Fetches curated PTB-XL records for AFIB, PVC, PSVT (AVNRT proxy), and WPW
2. Renders clean 12-lead ECG images (OpenCV, 300 DPI)
3. Applies clinical diagnosis overlays for anatomical labels not in PTB-XL
4. Writes PNGs and `arrhythmia_index.json` to this folder

### Generate via CLI (single types)

```bash
# AFIB
synthecg -n 1 -t AFIB --seed 42 --augment-profile clean --save-clean -o /tmp/afib

# PVC
synthecg -n 1 -t PVC --seed 42 --augment-profile clean --save-clean -o /tmp/pvc

# AVNRT (maps to PSVT in PTB-XL)
synthecg -n 1 -t AVNRT --seed 42 --augment-profile clean --save-clean -o /tmp/avnrt

# WPW
synthecg -n 1 -t WPW --seed 42 --augment-profile clean --save-clean -o /tmp/wpw
```

## Limitations

| Requested label | PTB-XL support | SynthECG approach |
|-----------------|----------------|-------------------|
| AFIB | ✅ `AFIB` code (1,514 records) | Direct filter `--type AFIB` |
| PVC (left coronary cusp) | ✅ `PVC` code; ❌ no anatomy | PVC record + overlay label |
| AVNRT | ❌ no `AVNRT` code | Alias to `PSVT` (24 records) + overlay label |
| WPW (right free wall) | ✅ `WPW` code (79 records); ❌ no pathway side | WPW record + overlay label |

Anatomical localization (coronary cusp, accessory pathway wall) requires electrophysiology mapping data not present in PTB-XL. The overlay labels document the intended clinical context for teaching and demo purposes.

## Image properties

All examples use:

| Property | Value |
|----------|-------|
| Layout | 3×4 + Lead II rhythm strip |
| Renderer | OpenCV (300 DPI) |
| Paper speed | 25 mm/s |
| Gain | 10 mm/mV |
| Preprocessing | 0.5–40 Hz bandpass filter |
