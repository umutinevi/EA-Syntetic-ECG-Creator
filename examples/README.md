## Regenerate arrhythmia examples

```bash
pip install -e .
python scripts/generate_arrhythmia_examples.py
```

This script:
1. Uses **Zheng EP-validated** records for PVC LCC
2. Uses PTB-XL for AFIB, PSVT (AVNRT proxy), and WPW
3. Runs literature-based localization algorithms where needed
4. Writes PNGs and provenance-aware `arrhythmia_index.json`

The PVC example requires the Zheng OT-VA ECG cache (~694 MB download on first run).

### Generate via CLI

```bash
# AFIB (PTB-XL)
synthecg -n 1 -t AFIB --seed 42 --augment-profile clean --save-clean -o /tmp/afib

# PVC with EP-validated LCC localization (Zheng)
synthecg -n 1 -t LCC --database zheng-otva --augment-profile clean -o /tmp/pvc_lcc

# Balanced localization dataset
synthecg dataset build --recipe arrhythmia-localization-v1 -o otva_localization

# AVNRT (maps to PSVT in PTB-XL)
synthecg -n 1 -t AVNRT --seed 42 --augment-profile clean -o /tmp/avnrt

# WPW with algorithm-inferred localization
synthecg -n 1 -t WPW --seed 42 --augment-profile clean -o /tmp/wpw
```

## Image properties

All examples use:

| Property | Value |
|----------|-------|
| Layout | 3×4 + Lead II rhythm strip |
| Renderer | OpenCV (300 DPI) |
| Paper speed | 25 mm/s |
| Gain | 10 mm/mV |
| Preprocessing | 0.5–40 Hz bandpass filter |
